# コマンド一覧(iidx 用)

| コマンド | 内容 | 対応レベル |
|---------|------|------------|
| dp_level | DP非公式難易度取得 | 12 |
| iidx | 課題曲生成 | 11,12 |

## \dp_level {曲名の一部}
- 曲名の一部を入力すると，楽曲の候補が表示される <br>

    ```text
  (例)
  [1] musicA 
  [2] musicB
  [3] musicC

- 想定した曲名の番号を送ることで，その楽曲のDP非公式難易度をbotが返す
- 現状はレベル12のみに対応


## \iidx {レベル(11 or 12)} {曲数}
- 指定したレベルから指定した曲数分の譜面を，ランダムでbotが選択し返信する
- 現在はレベル11,12のみに対応
- コマンド名は変更を検討中
